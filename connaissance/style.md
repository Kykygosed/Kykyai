# Mes conventions de code

## Général
- Je commente toujours mon code en français
- J'utilise des noms de variables clairs et descriptifs
- Je préfère les fonctions courtes et lisibles

## JavaScript
- J'utilise const/let, jamais var
- Arrow functions pour les callbacks
- Async/await plutôt que .then()

## CSS
- Je nomme mes classes en kebab-case
- J'utilise des variables CSS (--ma-variable)

## HTML
- Toujours un DOCTYPE
- Balises sémantiques (header, main, section, footer)
